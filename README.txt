Hand-To-Hand
Мой сайт предназначен для людей, которым нужна какая-то вещь на один раз, и им нет смысла её покупать, здесь они могу взять её в аренду. Для этого они могут либо найти уже из предложенных пользователями объявлений, либо создать объявление самому, чтобы их могли найти люди который хотят сдать свою вещь в аренду. Далее связаться с арендодателем и обговорить в сообщениях или по телефону.
Сейчас на сайте можно :
1. Загерестрировать нового пользователя.
2. Войти в существующий аккаунт.
3. Подать Объявление.
4. Посмотреть объявления других пользователей.
Сайт разработан с использованием языков HTML, SQL, Python и библиотеки Flask.
Сейчас сайт имеет вресию 1.0

4. Возможность оставлять и смотреть отзывы о других пользователях.
5. Просмотр профиля другого пользователя
